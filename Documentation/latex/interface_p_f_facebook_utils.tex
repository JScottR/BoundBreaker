\subsection{Detailed Description}
The {\ttfamily \hyperlink{interface_p_f_facebook_utils}{P\+F\+Facebook\+Utils}} class provides utility functions for working with Facebook in a \hyperlink{interface_parse}{Parse} application.

This class is currently for i\+O\+S only.

The {\ttfamily \hyperlink{interface_p_f_facebook_utils}{P\+F\+Facebook\+Utils}} class provides utility functions for using Facebook authentication with $<$\+P\+F\+User$>$s.

\begin{DoxyWarning}{Warning}
This class supports official Facebook i\+O\+S S\+D\+K v4.\+0+ and is available only on i\+O\+S. 
\end{DoxyWarning}
