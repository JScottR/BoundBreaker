\subsection{Detailed Description}
The {\ttfamily \hyperlink{interface_p_f_user}{P\+F\+User}} class is a local representation of a user persisted to the \hyperlink{interface_parse}{Parse} Data. This class is a subclass of a $<$\+P\+F\+Object$>$, and retains the same functionality of a $<$\+P\+F\+Object$>$, but also extends it with various user specific methods, like authentication, signing up, and validation uniqueness.

Many A\+P\+Is responsible for linking a {\ttfamily \hyperlink{interface_p_f_user}{P\+F\+User}} with Facebook or Twitter have been deprecated in favor of dedicated utilities for each social network. See $<$\+P\+F\+Facebook\+Utils$>$, $<$\+P\+F\+Twitter\+Utils$>$ and $<$\+P\+F\+Anonymous\+Utils$>$ for more information.
