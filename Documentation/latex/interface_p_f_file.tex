\subsection{Detailed Description}
{\ttfamily \hyperlink{interface_p_f_file}{P\+F\+File}} representes a file of binary data stored on the \hyperlink{interface_parse}{Parse} servers. This can be a image, video, or anything else that an application needs to reference in a non-\/relational way.
