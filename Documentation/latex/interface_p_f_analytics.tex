\subsection{Detailed Description}
{\ttfamily \hyperlink{interface_p_f_analytics}{P\+F\+Analytics}} provides an interface to \hyperlink{interface_parse}{Parse}\textquotesingle{}s logging and analytics backend.

Methods will return immediately and cache the request (+ timestamp) to be handled \char`\"{}eventually.\char`\"{} That is, the request will be sent immediately if possible or the next time a network connection is available.
